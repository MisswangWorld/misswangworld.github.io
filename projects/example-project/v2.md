# Example Project - v2

This release brings significant improvements and new features!

## Overview

Version 2 builds upon the foundation of v1 with enhanced performance, new features, and bug fixes based on user feedback.

## What's New in v2

### New Features

- **Dark Mode Support**: Toggle between light and dark themes
- **Improved Performance**: 50% faster load times
- **Mobile Optimization**: Better responsive design for all screen sizes
- **Export Functionality**: Export your data in multiple formats (JSON, CSV, PDF)

### Improvements

- Redesigned user interface for better usability
- Enhanced error handling and user feedback
- Updated dependencies for security patches

## Tech Stack

| Technology | Purpose |
|------------|---------|
| HTML/CSS   | Frontend structure and styling |
| JavaScript | Interactive functionality |
| Node.js    | Backend server |
| Redis      | Caching layer (new!) |

## Migration Guide

If you're upgrading from v1, follow these steps:

1. Backup your data
2. Update your configuration file
3. Run the migration script

```bash
npm run migrate:v2
```
