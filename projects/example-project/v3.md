## Getting Started

```bash
# Clone and install
git clone https://github.com/username/example-project.git
cd example-project
npm install

# Set up environment
cp .env.example .env
# Edit .env with your configuration

# Run development server
npm run dev
```

## API Example

```javascript
import { ExampleClient } from 'example-project-sdk';

const client = new ExampleClient({
  apiKey: 'your-api-key'
});

// Use natural language search
const results = await client.search('find all items from last week');

// Get AI suggestions
const suggestions = await client.getSuggestions();
```
